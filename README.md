# sales-data-analysis
Simple sales analysis project using Excel

## Objective
Analyze store sales data to identify top-selling products, best months, and highest-performing regions.

## Tools
- Microsoft Excel

## Dataset
Sales transaction dataset created for learning purposes.

## Analysis Questions
1. Which product has the highest total sales?
2. Which month has the highest sales?
3. Which region contributes the most revenue?
4. Which category generates the most sales?

## Insights
- Keyboard is the top-selling product.
- March has the highest sales.
- Jakarta contributes the most revenue.
- Electronics category generates the highest sales.

## Recommendations
- Increase stock for electronic products.
- Focus marketing on high-performing regions.
- Add more variations of best-selling products.
